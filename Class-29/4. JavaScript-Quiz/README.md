# JavaScript Quiz - 25

### 1. What is JavaScript?
A. Programming Language  
B. Markup Language  
C. Styling Language  
D. Database Language  
**Answer: A**

### 2. Which keyword is used to declare a variable?
A. var  
B. int  
C. string  
D. define  
**Answer: A**

### 3. Which keyword is block-scoped?
A. var  
B. let  
C. global  
D. define  
**Answer: B**

### 4. Which keyword is used for constants?
A. var  
B. let  
C. const  
D. static  
**Answer: C**

### 5. Which symbol is used for comments?
A. //  
B. `<!-- -->`  
C. ##  
D. **  
**Answer: A**

### 6. Which method prints to console?
A. print()  
B. console.log()  
C. log()  
D. echo()  
**Answer: B**

### 7. Which operator is used for strict equality?
A. ==  
B. ===  
C. =  
D. !=  
**Answer: B**

### 8. Which data type is NOT primitive?
A. string  
B. number  
C. object  
D. boolean  
**Answer: C**

### 9. Which function converts JSON to object?
A. JSON.parse()  
B. JSON.stringify()  
C. JSON.convert()  
D. JSON.toObject()  
**Answer: A**

### 10. Which function converts object to JSON?
A. JSON.parse()  
B. JSON.stringify()  
C. JSON.convert()  
D. JSON.toString()  
**Answer: B**

### 11. Which keyword defines a function?
A. func  
B. function  
C. def  
D. method  
**Answer: B**

### 12. Which loop runs at least once?
A. for  
B. while  
C. do...while  
D. foreach  
**Answer: C**

### 13. Which array method adds element at end?
A. push()  
B. pop()  
C. shift()  
D. unshift()  
**Answer: A**

### 14. Which array method removes last element?
A. push()  
B. pop()  
C. shift()  
D. unshift()  
**Answer: B**

### 15. Which method is used to select element by id?
A. getElement()  
B. getElementById()  
C. queryId()  
D. selectId()  
**Answer: B**

### 16. Which method selects all matching elements?
A. querySelectorAll()  
B. getAll()  
C. selectAll()  
D. getElements()  
**Answer: A**

### 17. Which event occurs when user clicks?
A. onhover  
B. onclick  
C. onchange  
D. onload  
**Answer: B**

### 18. Which method is used to delay execution?
A. setTimeout()  
B. wait()  
C. delay()  
D. timeout()  
**Answer: A**

### 19. Which keyword refers to current object?
A. self  
B. this  
C. object  
D. current  
**Answer: B**

### 20. Which operator spreads array/object?
A. ...  
B. ===  
C. =>  
D. **  
**Answer: A**

### 21. Which syntax defines arrow function?
A. function => {}  
B. () => {}  
C. => function()  
D. func() =>  
**Answer: B**

### 22. Which method is used to fetch API?
A. get()  
B. request()  
C. fetch()  
D. api()  
**Answer: C**

### 23. Which keyword handles errors?
A. catch  
B. error  
C. handle  
D. try  
**Answer: A**

### 24. Which statement is used with try?
A. catch  
B. finally  
C. Both A and B  
D. error  
**Answer: C**

### 25. Which method converts string to integer?
A. parseInt()  
B. toInt()  
C. Number()  
D. Both A and C  
**Answer: D**

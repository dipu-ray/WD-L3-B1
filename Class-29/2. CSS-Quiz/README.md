# CSS Quiz - 25

### 1. What does CSS stand for?
A. Creative Style Sheets  
B. Cascading Style Sheets  
C. Computer Style Sheets  
D. Colorful Style Sheets  
**Answer: B**

### 2. Which HTML tag is used to define internal CSS?
A. `<css>`  
B. `<script>`  
C. `<style>`  
D. `<link>`  
**Answer: C**

### 3. Which property is used to change text color?
A. font-color

D. fgcolor  
**Answer: B**

### 4. Which property is used to change background color?
A. bgcolor  
B. background-color

D. bg-color  
**Answer: B**

### 5. How do you select an element with id="demo"?

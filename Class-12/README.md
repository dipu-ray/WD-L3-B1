### 1. CSS Advanced
- `CSS Shadows`:
    - `text-shadow`
    - `box-shadow` - This Horizontal and Vertical Shadow
    - `box-shadow: 5px 5px 8px blue, 10px 10px 8px red, 15px 15px 8px green;` - Add Multiple Shadows
- `CSS Text Effects`:
    - `text-overflow` - Specifies how to handle overflowed content.
    - `word-wrap` - Allows long words to be able to be broken and wrap onto the next line.
    - `word-break` - Specifies line breaking rules.
    - `writing-mode` - Specifies whether lines of text are laid out horizontally or vertically.
    - `white-space: nowrap;`
    - `overflow: hidden;`
- `CSS drop-shadow() Function`:
    - `filter: drop-shadow(0 0);`

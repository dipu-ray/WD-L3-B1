# Educational Website

* [Back to Top ⬆️](#topics)

This **Educational Website** have about section, programs section, events section, gallery section and review section and JavaScript using for **Date and Time show**, **Form Validation** & **Scroll to Top Button**.

## 🛠️ Tech Stack

- **HTML:** Semantic structure.
- **CSS:** Colorful and style.
- **Bootstrap:** Responsive layout & prebuilt UI components.
- **JavaScript:** DOM manipulation and intervals.


# HTML (30 Questions)

* [Short Questions and MCQs](#short-questions-and-mcqs)

## 1. What does HTML stand for?
**Answer:** HyperText Markup Language.

## 2. What is the basic structure of an HTML document?
**Answer:**
```html
<!DOCTYPE html>
<html>
<head>
<title></title>
</head>
<body>
</body>
</html>
```

## 3. Which tag is used to create a hyperlink?
**Answer:** `<a>`

## 4. What is the purpose of the `<alt>` attribute in the `<img>` tag?
**Answer:** Provides alternative text for images.

## 5. How do you create an ordered list in HTML?
**Answer:** Use `<ol>` with `<li>` elements.

## 6. Difference between `<div>` and `<span>`?
**Answer:** `<div>` is block-level, `<span>` is inline.

## 7. Largest heading tag?
**Answer:** `<h1>`

## 8. What does `<br>` do?
**Answer:** Inserts a line break.

## 9. How to create a table?
**Answer:** Use `<table>`, `<tr>`, `<th>`, `<td>`.

## 10. Purpose of `<form>`?
**Answer:** Creates a form for user input.

## 11. Name two input types.
**Answer:** text, password, email, checkbox, radio.

## 12. Which attribute specifies image URL?
**Answer:** `src`

## 13. What is semantic HTML?
**Answer:** Using meaningful tags like `<header>`, `<section>`, `<footer>`.

## 14. How to write comments?
**Answer:**
```html
<!-- Comment -->
```

## 15. Purpose of `required` attribute?
**Answer:** Makes form fields mandatory.

## 16. Which tag defines webpage title?
**Answer:** `<title>`

## 17. What does `target="_blank"` do?
**Answer:** Opens link in a new tab.

## 18. How to embed video?
**Answer:** Use `<video>` tag.

## 19. Difference between `<strong>` and `<b>`?
**Answer:** `<strong>` has semantic importance, `<b>` only bolds text.

## 20. Root element of HTML?
**Answer:** `<html>`

## 21. Purpose of `<meta>`?
**Answer:** Provides metadata.

## 22. Tag for italic text?
**Answer:** `<em>` or `<i>`

## 23. How to create dropdown list?
**Answer:** Use `<select>` and `<option>`.

## 24. Purpose of `id`?
**Answer:** Unique identifier.

## 25. Purpose of `class`?
**Answer:** Groups elements for CSS/JS.

## 26. Name two HTML5 elements.
**Answer:** `<header>`, `<footer>`

## 27. Paragraph tag?
**Answer:** `<p>`

## 28. How to link CSS?
**Answer:**
```html
<link rel="stylesheet" href="style.css">
```

## 29. Default display of `<img>`?
**Answer:** Inline.

## 30. Name one form validation attribute.
**Answer:** `required`

# CSS (30 Questions)

## 31. What does CSS stand for?
**Answer:** Cascading Style Sheets.

## 32. How to link external CSS?
```html

```

## 33. CSS syntax?
```css
selector {
  property: value;
}
```

## 34. Difference between id and class selector?
**Answer:** `#id` unique, `.class` reusable.

## 35. Property for text color?
**Answer:** `color`

## 36. Make text bold?
```css

```

## 37. What does `margin: 0 auto;` do?
**Answer:** Centers block element horizontally.

## 38. CSS box model?
**Answer:** Content → Padding → Border → Margin.

## 39. Prevent background repeat?
```css
background-repeat: no-repeat;
```

## 40. Property for inner spacing?
**Answer:** `padding`

## 41. Difference between block and inline?
**Answer:** Block starts new line, inline stays in same line.

## 42. Responsive image CSS?
```css
max-width: 100%;
height: auto;
```

## 43. Purpose of `position: relative`?
**Answer:** Positions relative to normal position.

## 44. Pseudo-class for hover?
**Answer:** `:hover`

## 45. Flex container property?
```css

```

## 46. Media query for smaller screens?
```css

}
```

## 47. Purpose of `z-index`?
**Answer:** Controls stacking order.

## 48. Center text?
```css

```

## 49. Font family property?
**Answer:** `font-family`

## 50. What does `border-radius` do?
**Answer:** Rounds corners.

## 51. Hide element but keep space?
```css

```

## 52. Difference between em and rem?
**Answer:** em = parent relative, rem = root relative.

## 53. Property for transparency?
**Answer:** `opacity`

## 54. What does `float: left;` do?
**Answer:** Floats element left.

## 55. Make list horizontal?
**Answer:** `display: inline-block;` or flex.

## 56. Universal selector?
**Answer:** `*`

## 57. Remove underline from links?
```css
text-decoration: none;
```

## 58. Background color property?
**Answer:** `background-color`

## 59. Name one animation property.
**Answer:** `animation-duration`

## 60. What does `flex-wrap: wrap;` do?
**Answer:** Allows wrapping to new lines.
